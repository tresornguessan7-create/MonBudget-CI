# MonBudget-CI
MonBudget CI est une application web de gestion budgétaire personnelle en FCFA, pour salariés et freelances en Côte d'Ivoire et dans la zone FCFA. Avec cette appli, vous pourrez planifier votre budget, tracker vos dépenses quotidiennes et analyser votre bilan mensuel. C'est gratuit, sans inscription et vos données restent sur votre téléphone.
